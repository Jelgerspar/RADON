Introduction Last updated: 2020-7-15

Radon is an open-source, decentralised cryptocurrency with price increase features. Also you can run DAPP's on Radon (DAPP's are decentralised applications). On this page, we will teach you how to use your browser to interact with the smart contract of Radon.

1.1 Why Choose Radon

You should choose to build your D'App on top of the Radon ecosystem, because you believe in the 20 Phases plan, and want to level-up your brand. Although Radon isn't the biggest currency out there, there are real humans using this coin. If you want to help these people out with a D'app or want to capitalize on the Radon invention, you should build on top of us.

Web3.js - Radon Blockchain integration on the web

Because Radon is build on top of the Ethereum blockchain, developers can use Metamask to make Radon transactions or integrate other Web3.js features in their website.

While testing your project, you should always run a localhost or alternatives, because of CORS policies.

2.1 Check for Web3.js Support

To check if a user has a web3.js provider installed, you have to check for the window.ethereum variable.

Remember to run a localhost or use alternatives!


After you checked for a web3 provider you may want to connect with the extention, you do this with the ethereum.enable function.

If the user accepts your request, you have access to his/her account and you can start interacting with the Radon smart contract.

As you can see, we use the connectWeb3() async function, because you wait for the user to accept or decline your transaction request.
